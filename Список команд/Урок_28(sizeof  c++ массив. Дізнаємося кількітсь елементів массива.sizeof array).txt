в sizeof ми можемо підставити змінну або тип даних. 

int arr[]{5, 567, 78, 35, 597};
for (int i = 0; i < sizeof(arr) / sizeof(int); i++)
	{
		cout << arr[i] << endl;
	}

в цьому коді ми sizeof(arr) / sizeof(int) ми дізналися звідси скільки елементів в array. в sizeof(arr) у нас 20 байт, тому що 1 елемент дорівнює 4 байтам, а там в нас лежить їх 5, ну і в sizeof(int)  у нас дорівнює 4 байтам, тому що тип даних int має 4 байти. Ну і трохи математики 20 / 4 = 5- елементів в массиві.

///////////////////////////////////


	double arr[]{5, 567, 78, 35, 597};

	for (int i = 0; i < sizeof(arr) / sizeof(arr[0]); i++)
	{
		cout << arr[i] << endl;
	}

в цьому коді самий найкращий спосіб дізнатися кількість елементів в массиві.
sizeof(arr[0]) - він бере 0 елемент і визначає його тип по байтам, а отже якщо ввести любий тип даних, програма коректно буде працювати навіть якщо внести якісь змінні в тип даних або зменшити або збільшити кількість елементів і тд.
