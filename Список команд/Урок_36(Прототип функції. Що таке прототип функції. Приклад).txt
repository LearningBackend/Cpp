Для того щоб функцію можна було використовувати в main навіть якщо вона була пізніше прописанна, ми можемо її просто реалізувати таким чином:

void foo(); <----------ми тут просто реалізуємо її, але не описуємо.


void main()
{
	SetConsoleCP(1251);
	SetConsoleOutputCP(1251);
	
	foo();

}


void foo()
{
	cout << "Я функція і мене визвали" << endl;
}

прототип функції є передварним оголошенням функції. Він дозволяє компілятору знати про існування функції та її сигнатуру (ім'я, типи параметрів і тип повернення) перед тим, як фактичне тіло функції буде визначено.

Прототип функції містить лише інформацію про її ім'я, типи параметрів і тип повернення, без самого коду функції. Це корисно в ситуаціях, коли функції викликаються перед їхнім визначенням у файлі або коли функції визначені в іншому файлі чи бібліотеці
