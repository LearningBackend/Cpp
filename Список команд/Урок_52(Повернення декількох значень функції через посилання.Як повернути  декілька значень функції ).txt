void Foo(int& a, int& b, int& c)
{
	a = 10;

	b *= 2;

	c -= 100;

}

void main()
{
	SetConsoleCP(1251);
	SetConsoleOutputCP(1251);

	int a = 0, b = 4, c = 34;
	cout << "a = " << a << endl;
	cout << "b = " << b << endl;
	cout << "c = " << c << endl;

	cout << "Foo" << endl;
	
	Foo(a, b, c);

	cout << "c = " << c << endl;


}

У C++ функція може повертати лише одне значення, але ви можете використовувати посилання або покажчики для "повернення" декількох значень через параметри функції. Ось коротка теорія щодо цього:

Повернення декількох значень функції через посилання або покажчики:

Ви можете передавати декілька параметрів у функцію за посиланням або через покажчики, і зміни в цих параметрах будуть відображатися за межами функції.
Це дає можливість функції змінювати значення кількох змінних, "повертаючи" їхні зміни через параметри.
