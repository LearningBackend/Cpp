Логічні операції:

1) ОПЕРАТОРИ ПОРІВННЯ

< - менше
 
> - більше

<= - менше або дорівнює

>= - більше або дорівнює

2) ОПЕРАТОРИ РІВНОСТІ

== - дорівнює або true або 1

!= - не дорівнює або false або 0

3)ОПЕРАТОРИ ОБ'ЄДНАННЯ І НЕГАТИВНА ІНВЕРСІЯ

&& - і

|| - або

!= - ні 


//////////////////////

інверсування, тобто зміннює результат на протилежний
наприклад

int а = 3, b = 3;
cout << !(a == b) << endl;

отже в нас тут є ! який змінює(інвертує результат) в консоль кінцевий результат. тобто 3 = 3 і має вивести нам 1(true), але виведе нам 0(false) тому що знак !(інверсія) змінює результат на протилежний.  

////////////////////////

cout << ((a == b) && (5 < 4)) << endl;

В цьому коді компілятор перевіряє чи правильно всі умови записанні, і якщо одна з них не вірна нам в консоль повернеться 0
