ASCII (American Standard Code for Information Interchange) - це стандарт кодування символів, який використовується для представлення текстової інформації в комп'ютерах і інших пристроях. ASCII визначає числовий код для кожного основного символу, які можуть бути використані в тексті, такі як літери, цифри, розділові знаки та спеціальні символи. Це стандартна система кодування, яка дозволяє комп'ютерам розуміти та обробляти текстову інформацію.

Таблиця ASCII включає 128 символів, від 0 до 127, і відображає їхні числові коди. 
Кожен символ у таблиці ASCII відповідає певному числовому значенню, яке можна використовувати в програмах для представлення і обробки текстової інформації. В мові програмування C++ можна використовувати числові коди ASCII для роботи з символами. 

SetConsoleCP(1251);
SetConsoleOutputCP(1251);
for (int i = 0; i < 255; i++)
{
	cout << "code = " << i << " " << "char = " << (char)i << endl;
}
