Конкатенація рядків (або об'єднання рядків) - це операція об'єднання двох або більше рядків в один. Ця операція зазвичай використовується для створення нового рядка, який містить символи з об'єднаних рядків. Конкатенація рядків може бути корисною для створення більш складних рядків або форматування тексту.

У мові програмування C++, для роботи з рядками та їх конкатенації зазвичай використовуються об'єкти класу string.
string надає багато зручних методів для роботи з рядками, таких як append, substr, find, і багато інших, що полегшують обробку рядків в C++. Важливою перевагою string є те, що він автоматично керує пам'яттю і забезпечує безпеку роботи з рядками, уникнення переповнення буфера і інших типових проблем, пов'язаних з роботою з масивами char вручну.

Конкатенація код 

char result[255]{};
char str1[255] = "Hello";
char str2[255] = "World";

cout << str1 << endl;

strcat_s(result, str1);
strcat_s(result, str2);

cout << result << endl;

string str1 = "Hello";
string str2 = "World";
string str3 = "!!!";
string result;

result = str1 + " " + str2 + str3;
