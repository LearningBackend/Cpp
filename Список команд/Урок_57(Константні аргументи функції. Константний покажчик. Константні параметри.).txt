Константні аргументи, константний покажчик і константні параметри в C++ використовуються для обмеження можливостей зміни даних, передачі параметрів та вказівників. Ось коротка теорія щодо цих понять:

Константні аргументи функції (const arguments):
Константні аргументи - це аргументи функції, які оголошені як константні. Тобто значення цих аргументів не можна змінювати всередині функції. Це допомагає забезпечити намірену поведінку функції і зберегти незмінність даних

Константні покажчики (const pointers):
Константний покажчик - це покажчик, який вказує на константні дані із забороною зміни значення, на яке він вказує. Константний покажчик може бути вказівником на змінну або об'єкт, значення якого не можна змінювати, але сам покажчик може вказувати на інші об'єкти

Константні параметри функції (const parameters):
Константні параметри - це параметри функції, які оголошені як константні. Це означає, що функція не може змінювати значення цих параметрів. Це корисно для захисту вхідних даних від випадкової модифікації.

Константні аргументи, покажчики та параметри є важливою частиною C++, оскільки вони допомагають забезпечити безпеку та контроль над даними, запобігаючи їх незапланованій зміні.

void FillArray(int* const arr, const int size)
{
	for (int i = 0; i < size;  i++)
	{
		arr[i] = rand() % 10;
	}
}

void ShowArray(const int* const arr, const int size)
{
	for (int i = 0; i < size; i++)
	{
		cout << arr[i] << "\t";
	}
	cout << endl;
}
