Тема: Рядки і нуль-термінатор у C++

1. Рядок у C++

Рядок у C++ представляє собою послідовність символів, які завершуються нуль-термінатором ('\0').
Рядки можна представити у вигляді масиву символів (char) із завершальним нуль-термінатором.
2. Масив char

Масив char у C++ - це звичайний масив, що містить символи. Наприклад: char myString[10];.
Символи у масиві char індексуються від 0 до n-1, де n - довжина масиву.
3. Нуль-термінатор

Нуль-термінатор - це спеціальний символ '\0', який позначає кінець рядка.
Всі рядки у C++ завершуються нуль-термінатором, щоб вказати кінець рядка.
4. Ініціалізація рядків

Рядки можна ініціалізувати різними способами:
char myString[] = "Привіт"; - компілятор автоматично вставить нуль-термінатор.
char myString[10] = "Привіт"; - вказуємо розмір масиву.
5. Робота з рядками

Для виконання операцій над рядками в C++ використовуються функції з бібліотеки <cstring>:
strlen(str) - повертає довжину рядка без нуль-термінатора.
strcpy(dest, src) - копіює рядок src в рядок dest.
strcmp(str1, str2) - порівнює два рядки str1 і str2.
і багато інших.
6. Звернення до символів рядка

Для звернення до окремих символів рядка використовують індексацію: myString[i], де i - індекс символу.
7. Завершення роботи з рядком

Важливо пам'ятати, що масив char має бути достатньо великим, щоб вмістити рядок разом із нуль-термінатором.
Всі операції над рядками повинні бути обережними, щоб уникнути виходу за межі масиву та переписування пам'яті.
