Передача параметрів у функцію за посиланням:

При передачі параметра за посиланням, функція отримує доступ до самого об'єкта, а не його копії.
Зміни, внесені до параметра в середині функції, впливатимуть на оригінальний об'єкт.
Різниця між посиланням і покажчиком:

Посилання - це альтернативна назва для існуючої змінної; його потрібно ініціалізувати при створенні і не можна перенаправити на інший об'єкт.
Покажчик - це змінна, яка містить адресу іншої змінної; покажчики можна перенаправляти на інші об'єкти.

void Foo(int a)
{
	a = 1;
}

void Foo2(int &a)
{
	a = 2;
}

void Foo3(int *a)
{
	*a = 3;
}

void main()
{
	SetConsoleCP(1251);
	SetConsoleOutputCP(1251);

	int value = 5;
	cout << value << endl << endl;
	
	cout << "Foo = " << endl;
	Foo(value);
	cout << "value = " << value << endl << endl;

	cout << "Foo2 = " << endl;
	Foo2(value);
	cout << "value = " << value << endl << endl;

	cout << "Foo3 = " << endl;
	Foo3(&value);
	cout << "value = " << value << endl << endl;
}
