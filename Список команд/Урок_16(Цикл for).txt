Цикл for - це останній вид циклів, зручний при роботі з масивами.

записується так:

for (int i = 0; i < length; i++)
	{

	}

int i = 0; - (ініціалізуємо і(надаємо їй значення))

i < length; - це умова циклу, тобто якщо ця функція повертає true, то цикл продовжує виконуватися, а якщо false, то ні.

i++; - інкремент змінної, збільшується на 1.
