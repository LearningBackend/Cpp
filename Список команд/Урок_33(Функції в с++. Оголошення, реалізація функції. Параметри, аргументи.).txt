Функції в с++ - це конструкція в програмуванні яка дозволяє винести деякий часто повторювальний участок коду, десь окремо, а потім щоб знову не писати цей код, ми можемо замість нього визвати функцію і вона виконає тей код.

Синтаксис функції:

повертаюче_значення ім'я_функції (налаштування)
{
блок_повторюючого_коду (тіло);
}

Приклад:
void foo()
{
	cout << "Я функція, мене визвали!" << endl;
}

Цікавий момент, функція яку нам потрібно визвати має бути вище, чим функція в якій ми хочемо її визвати. Але цю проблему можна вирішити якшо заморочитись, дізнаюся далі як це зробити.

Return функція:

це функція яка повертає значення, ось як використовується:
short Sum(short a, short b) 
{
	short result;
	result = a + b;
	return result;
}
void main()
{
	SetConsoleCP(1251);
	SetConsoleOutputCP(1251);

	short c;

	c = Sum(2, 5);

	cout << c << endl;
	
}

якщо нам не потрібно ніде зберігати результат функції, ми можемо зробити так:

cout << Sum(q, w) << endl;
