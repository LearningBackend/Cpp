NULL та nullptr:

NULL - це макрос, який використовується для позначення нульового покажчика (непризначеного покажчика).
nullptr - це спеціальне ключове слово в C++11 і пізніших стандартах для позначення нульового покажчика. Воно більш безпечне і зручне в порівнянні з NULL.
Робота з пам'ятю та нульовими покажчиками:

Покажчики, що не мають призначення (нульові покажчики), не вказують на жоден об'єкт або адресу.
Використання нульових покажчиків може призвести до помилок, які називаються "нульовими деференсами" (при спробі розіменування нульового покажчика).
Використання nullptr:

Використовуючи nullptr у місці NULL, ви можете уникнути деяких можливих проблем з нульовими покажчиками.
nullptr можна використовувати у умовах, при порівнянні з покажчиками та виклику функцій.

int* pa = new int;
	*pa = 10;
	cout << *pa << endl;
	delete pa;

	pa = nullptr;

	if (pa != nullptr)
	{
		cout << pa << endl;
	}
	
	delete pa;
