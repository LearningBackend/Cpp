Умовний тернарний оператор - це спеціальна конструкція в багатьох програмувальних мовах, включаючи C++, яка дозволяє виразити умовну логіку компактним способом. Основна ідея полягає в тому, що ви можете вибрати один із двох можливих значень виразу в залежності від заданої умови.

Умовний тернарний оператор особливо зручний для коротких умовних виразів та призначений для підвищення читабельності коду, зменшення кількості рядків і спрощення виразів у тих випадках, коли вам потрібно вибирати одне з двох значень залежно від умови.

if (a<10)
{
	cout << "а меньше 10" << endl;
}
else 
{
	cout << "а більше 10" << endl;
}

a<10 ? cout << "а меньше 10" << endl: cout << "а більше 10" << endl;
цей запис дорівнює запису if else

a<10 якщо він true то робиться відразу перша дія після ?

питаєм типа a < 10 істинна(? це і є знак питання)
